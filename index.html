Work I have done

PSD Conversion

The Restaurant Mall is a site that runs off of pure HTML & CSS showcasing my ability to receive an image (in this case a .psd file) and translate that image into a working fully functional website.

Find Me Some Food

Find Me Some Food was a pair programming project where myself and Christine Shiels worked together using jQuery and the MapQuest API to create a website showing the user the closest restaurants to their inputted address.

Weather App

I created this weather app using React & two APIs (WeatherAPI, UnsplashAPI) with the goal to keep the data and the UX as clean and simple as possible. The user inputs a city and the app will display the current weather & a nice city background, show a forecast if desired, and offer the ability to save that city for future use.
